Clean Energy Project

This project investigates where different type of energy sources are located in the United States and their capacity in megawatts (MW). It uses the World Resources Institute’s Global Power Plant Database and filters for U.S. facilities. This project's main output is an interactive Plotly bubble map for the United States with geographic clustering sorted by energy source. Other simple graphs are included for a further analysis of energy trends in the United States.

Problem

This project analyzes the geographic distribution and capacity of energy sources across the United States using the Global Power Plant Database. It aims to visualize patterns and trends in power generation through an interactive Plotly bubble map and supporting analyses.

Approach

The approach involves cleaning and filtering the Global Power Plant Database to isolate U.S. facilities and relevant variables. The data is then transformed and visualized using Plotly to create an interactive bubble map, along with additional plots to analyze trends in energy capacity and distribution.

Results & Impact

The results highlight clear geographic patterns and capacity differences among energy sources across the United States. These insights support better understanding of the current energy landscape and can inform future planning, policy decisions, and resource allocation.

Capstone - Kellanova Demand Forecasting Project

This capstone project focused on building and evaluating demand forecasting models for weekly point-of-sale sales data in collaboration with Kellanova. Because the original data is protected under an NDA, only a generic copy of the project using dummy data is shared. The project compares several forecasting approaches, including SARIMAX, Prophet, NBEATS, NHits, LSTM, XGBoost, Chronos2, TSMixer, LightGBM, and DeepAR. The analysis focuses on forecast accuracy, provider-level performance, bias, volatility, and model comparison across a 26-week hold-out period.

Problem

This project addresses the challenge of forecasting weekly consumer demand using historical point-of-sale data and trade-related features. The goal was to identify models that could accurately predict future sales patterns across multiple providers while accounting for trends, seasonality, promotional activity, and trade distribution effects. Since the original data is protected under an NDA, the shared version of the project uses dummy data to demonstrate the structure, workflow, and modeling process without exposing protected information.

Approach

The project cleaned and aggregated weekly sales data by provider, engineered time-based and trade activity features, and tested multiple forecasting models over a consistent 26-week hold-out period. Models were compared using standardized metrics such as RMSE, MAPE, WMAPE, accuracy, bias, volatility ratio, and standard deviation of absolute percentage error.

Results & Impact

The results provided a structured comparison of several forecasting methods and highlighted differences in model performance across providers. The project showed that demand forecasting accuracy can vary depending on provider behavior, sales volatility, and model type. By incorporating metrics beyond basic error measurements, such as bias and volatility ratio, the analysis helped identify models that were not only accurate on average but also more stable and reliable. These findings can support future forecasting workflows, model selection, and business planning for demand-driven decision making.

Data Note

The original project data is protected under a non-disclosure agreement, so the shared version of this project only includes dummy data. This allows the general project structure, notebooks, modeling workflow, environment setup, and visualization outputs to be reviewed without exposing confidential data.
